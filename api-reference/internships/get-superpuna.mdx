Retrieve a list of available internships from Superpuna. Our intelligent AI-powered scraper automatically extracts and processes internship listings from Superpuna's platform, using machine learning algorithms to ensure accurate data extraction.

## API Request

<CodeGroup>
	```bash curl -X GET
	https://scrape.fillonit.tech/api/get_internships/superpuna ```
</CodeGroup>

## Response

<ResponseField name="status" type="number">
	The HTTP status code of the response (200 for success)
</ResponseField>

<ResponseField name="data" type="array">
	Array of internship listings
	<Expandable title="Show array properties">
		<ResponseField name="title" type="string">
			The title of the internship position
		</ResponseField>
		<ResponseField name="company" type="string">
			The company offering the internship
		</ResponseField>
		<ResponseField name="location" type="string">
			The location of the internship
		</ResponseField>
		<ResponseField name="link" type="string">
			Direct URL to the internship posting
		</ResponseField>
		<ResponseField name="posted_date" type="string">
			The date when the internship was posted
		</ResponseField>
		<ResponseField name="duration" type="string">
			Expected duration of the internship
		</ResponseField>
		<ResponseField name="skills_required" type="array">
			List of required skills for the internship
		</ResponseField>
	</Expandable>
</ResponseField>
